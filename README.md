# Full-Stack Systems and Data Portfolio
Live Site: http://hishamn2.pythonanywhere.com

## Overview
This project is a production-ready portfolio application designed to bridge the gap between Systems Engineering and Data Science. It features a Flask backend, a relational SQLite database, and dynamic data visualizations.

## Tech Stack
- Backend: Flask (Python)
- Database: SQLite3
- Frontend: Bulma CSS, JavaScript (ES6+)
- Data Visualization: Chart.js (Radar Skill Matrix)
- Environment: Linux / WSGI Deployment

## Key Features
- Project Filtering: Custom JS logic to toggle between Systems and Data Science projects.
- Skill Matrix: Interactive Radar Chart quantifying technical and linguistic proficiencies (French, Arabic, English, Turkish).
- Contact System: Persistent SQL storage for user inquiries.
- Responsive Architecture: Mobile-first design optimized for all viewports.

## Installation and Setup
1. Clone the repository:
   git clone https://github.com/hishamn2/my_portfolio.git

2. Install dependencies:
   pip install -r requirements.txt

3. Initialize the database and run:
   python app.py
